Mobi Pick

Mobi Pick is an Android-style datepicker widget for jQuery Mobile. It uses the date library XDate. Mobi Pick allows progressive enhancement for date input fields using Modernizr.

If your browser has native support for date inputs, you should see these below. In any other browser, Mobi Pick will appear.

Feel free to take a look at the source code or fork this project on GitHub.
